## 本地运行

- clone项目源码

```bash
git clone git@github.com:Valdanitooooo/dgraph-docs-zh-cn.git
```

- 安装docsify-cli, 关于[docsify](https://docsify.js.org)

```bash
npm i docsify-cli -g
```

- 运行项目

在项目根目录下执行：

```bash
docsify serve ./docs
```

访问项目主页http://localhost:3000
